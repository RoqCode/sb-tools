# Storyblok Tools Monorepo

Small utilities for auditing Storyblok spaces and content. Each sub-project is self-contained.

## Projects

- [unused-components-finder](unused-components-finder/README.md): CLI to find (and optionally delete) unused Storyblok components in a space.
- [asset-audit](asset-audit/README.md): Node.js script to audit referenced Storyblok assets and flag oversize files across spaces.
